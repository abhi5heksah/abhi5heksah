# 🚀 Featured Projects

A detailed look at some of my flagship projects, demonstrating my expertise in full-stack web development, backend architecture, and interactive frontend design.

## 1. 🏢 Parlour Booking System

A comprehensive Full Stack platform enabling seamless online service booking, dynamic discount management, and administrative control.

- **Stack:** React.js, Node.js, Express, MongoDB, Tailwind CSS, GSAP
- **Key Features:**
  - **Google SSO Integration:** Implemented secure single sign-on for a frictionless user login experience.
  - **Dynamic Discount Engine:** Allowed admins to create real-time offers and discounts that automatically reflect on the user-facing platform.
  - **Dedicated Admin Dashboard:** Complete administrative control over user management, booking approval workflows, and service configuration.
  - **Interactive UI:** Enhanced user experience using React Bits and GSAP for micro-animations.

## 2. 🛡️ CompliSec (Hackathon Winner)

An automated vulnerability scanning tool capable of performing security assessments on target IP addresses and ports. Built during the Yukti Sangam Hackathon 2025.

- **Stack:** Python, Flask, Nmap, OpenVAS, Nikto
- **Key Features:**
  - **Multi-Level Scanning:** Implemented Basic, Intermediate, and Advanced scanning modes to support different levels of security analysis.
  - **Tool Integration:** Combined industry-standard tools for port scanning, service detection, and vulnerability identification.
  - **Security Reporting:** Generated detailed reports highlighting open ports, detected vulnerabilities, and potential security risks.
